# PajaritoHack 🐦

Proyecto de reconocimiento y clasificación de aves usando audio y datos de CSV.

## Descripción

PajaritoHack es una herramienta diseñada para identificar y clasificar especies de aves mediante análisis de audio.

## Características

- Análisis de archivos de audio de aves
- Clasificación automática de especies
- Procesamiento de datos ornitológicos
- Interfaz de usuario intuitiva

## Requisitos

- Python 3.7+
- Librerías especificadas en `requirements.txt`

## Instalación

```bash
git clone https://github.com/HernanVirgilioFerreyra/PajaritoHack.git
cd PajaritoHack
pip install -r requirements.txt
```

## Uso

```bash
python PajaritoHack.py
```

## Datos

El proyecto incluye:
- `datospajaros.csv`: Base de datos de especies de aves
- `audios/`: Conjunto de archivos de audio para análisis
